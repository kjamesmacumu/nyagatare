Description of software

The software has been developed by using R and its packages and by integrating R and JSP and web interface has been developed by using HTML,CSS and Java script

Programming in R

R is a programming language and software environment for statistical analysis, graphics representation and reporting. R was created by Ross Ihaka and Robert Gentleman at the University of Auckland, New Zealand, and is currently developed by the R Development Core Team.

The integration of R software with JSP

The integration of R Software with JSP using Netbeans IDE has been done by adding jar files into the libraries.

NetBeans IDE

NetBeans IDE is an open source integrated development environment dedicated to develop rock solid software solutions that address the need of developers and users
